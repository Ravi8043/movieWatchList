├── public/         # Public assets

## Installation

To install the Movie Watch List application, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/Ravi8043/movieWatchList.git
   ```
2. Navigate into the project directory:
   ```bash
   cd movieWatchList
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To run the Movie Watch List application locally:

1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to `http://localhost:3000`

You can now add, remove, and manage your movie watch list!
